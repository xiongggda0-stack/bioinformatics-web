# Roadmap

这里记录后续开发建议，按优先级排序。

## 当前阶段：公开展示型知识平台

- 安全扫描与示例脱敏。
- Pipeline 和软件详情页可信度面板。
- 面向研究问题的首页入口。
- 学习路径页。
- 搜索关键词高亮与热门入口。

## Phase 1：知识库体验增强

- Pipeline 列表搜索和筛选继续优化。
- Pipeline 详情页同类推荐继续增强。
- Algorithm 和 Literature 也加入搜索和筛选。
- 文档内容继续补充真实流程和图例。

## Phase 2：后台管理

目标：让用户可以在网页中新增和编辑内容。

建议功能：

- Pipeline 新增/编辑页面
- Algorithm 新增/编辑页面
- Literature 新增/编辑页面
- Markdown 编辑预览
- DAG JSON 可视化编辑

## Phase 3：用户系统

建议功能：

- 登录
- 用户角色
- 管理员权限
- 普通用户只读

## Phase 4：分析任务系统

这是从“知识库”进入“云平台”的关键阶段。

建议功能：

- 上传 FASTQ/表达矩阵
- 创建分析任务
- 任务状态追踪
- 日志展示
- 结果文件下载

## Phase 5：生产部署

建议功能：

- Alembic 数据库迁移
- Nginx 反向代理
- HTTPS
- CI/CD
- 生产环境变量管理
- 数据备份
